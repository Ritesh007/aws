# Exam

## Exam Format
- 3 hour exam
- 75 questions
- Plan no more than 2 minutes per question
- If it's a difficult question mark it for review and move on to the next

## Exam Topics
- SDLC automation (22%)
  - Topics such as CI/CD pipelines etc
- Configuration management and IaC (19%)
  - Cloudformation, EBS, Lambda, Opsworks etc
- Monitoring and Logging (15%)
- Policies and Standards automation (10%)
- Incident and Event Response (18%)
- High Availability, Fault Tolerance and Disaster Recovery (16%)
